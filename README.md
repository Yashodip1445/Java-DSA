# 🚀 Java Data Structures & Algorithms (DSA)

Welcome to my **Java DSA Repository** 💻
This repository contains all important **Data Structures & Algorithms concepts**, categorized into **Linear and Non-Linear Data Structures**, along with examples and problem-solving techniques.

---

## 📚 What is Data Structure?

A **Data Structure** is a way to organize and store data efficiently so that it can be accessed and modified easily.

---

## ⚡ Types of Data Structures

### 🔹 1. Linear Data Structures

Elements are arranged sequentially.

#### ✔ Examples:

* Array
* Linked List
* Stack
* Queue

---

### 🔹 2. Non-Linear Data Structures

Elements are arranged hierarchically.

#### ✔ Examples:

* Tree
* Graph
* Heap
* Trie

---

# 🧩 Linear Data Structures

## 📌 Array

* Stores elements in contiguous memory.
* Fast access using index.

```java
int[] arr = {1, 2, 3, 4, 5};
System.out.println(arr[0]); // 1
```

---

## 📌 Linked List

* Nodes connected using pointers.

```java
class Node {
    int data;
    Node next;
}
```

---

## 📌 Stack (LIFO)

* Last In First Out

```java
Stack<Integer> stack = new Stack<>();
stack.push(10);
stack.pop();
```

---

## 📌 Queue (FIFO)

* First In First Out

```java
Queue<Integer> q = new LinkedList<>();
q.add(10);
q.remove();
```

---

# 🌳 Non-Linear Data Structures

## 📌 Tree

* Hierarchical structure

```java
class TreeNode {
    int val;
    TreeNode left, right;
}
```

---

## 📌 Graph

* Collection of nodes and edges

```java
ArrayList<ArrayList<Integer>> graph = new ArrayList<>();
```

---

## 📌 Heap (Priority Queue)

* Used for min/max element

```java
PriorityQueue<Integer> pq = new PriorityQueue<>();
pq.add(10);
```

---

## 📌 Trie

* Used for string searching

```java
class TrieNode {
    TrieNode[] children = new TrieNode[26];
    boolean isEnd;
}
```
